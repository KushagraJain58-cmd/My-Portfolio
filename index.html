The Road Safety Protocol

AKA No Human Vehicle Tolling System

This prototype aims to minimize the number of Accidents occurring on roads due to Red-light Jumping, over speeding and Drunk and Drive cases using the combination of Hardware and Software. In addition, it would address the issues such as Emergency Lanes Misuse, Deaths in Ambulance due to traffic related delays. It Utilizes Arduino, ATMEGA-328, RFID, and many sensors as Hardware. As a part of Software it uses PHP, HTML, CSS, JavaScript & MySQL as database.

Tech Stack:
  • Arduino
  • C++
  • PHP
